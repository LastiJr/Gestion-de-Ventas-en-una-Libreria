INSTRUCCIONES DE INSTALACIÓN Y EJECUCIÓN

1. Descargar el proyecto
   - Ir al GitHub del grupo.
   - Presionar el botón "Code → Download ZIP".
   - Extraer el archivo en una carpeta de tu computador.

2. Instalar Java
   - Descargar e instalar Java JDK 8 o superior.
   - Verificar que funciona escribiendo en la terminal:
     java -version

3. Abrir el proyecto en un IDE
   - Recomendado: Eclipse, IntelliJ IDEA o NetBeans.
   - En el IDE, elegir "Abrir proyecto / Importar" y seleccionar la carpeta descargada.

4. Compilar el proyecto
   - Normalmente el IDE compila automáticamente.
   - Si no, compilar manualmente los archivos de la carpeta src/libreria.

5. Ejecutar la aplicación
   - Buscar la clase Sistema.java.
   - Presionar el botón "Run" (ejecutar).
   - También se puede ejecutar por consola con:
     java libreria.Sistema

6. Usar el menú
   - Al correr el programa aparecerá un menú en la consola.
   - Desde allí podrás:
     - Agregar libros
     - Crear estantes
     - Registrar clientes
     - Registrar ventas (descuenta stock)
     - Listar estantes y libros
